Ernährung noun, feminine—

nutrition n

Eine gesunde Ernährung beugt gegen Krankheiten vor. — Healthy nutrition prevents illnesses.
Gute Ernährung ist ein wesentliches Element eines gesunden Lebensstiles. — Good nutrition is an essential element of a healthy lifestyle.

diet n

Eine gute Ernährung ist vorteilhaft für die Gesundheit. — A good diet is beneficial to health.
Eine gesunde Ernährung erfordert hochwertige Nahrungsmittel. — A healthy diet requires high-quality food.
